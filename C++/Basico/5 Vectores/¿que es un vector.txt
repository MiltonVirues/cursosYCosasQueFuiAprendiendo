¿Que es un vector o un arreglo?
* Un Vector o Array unidimensional es un tipo de dato que almacena una secuencia de datos del mismo tipo.
Los elementos de un vector se almancenan en zonas contiguas de memoria y se puede acceder a ellos de manera directa mediante un indice o posicion.

Para Definir un Vector se utiliza:

tipo de nombre [tamaño];


Ejemplos:

 int numeros[10];
char letras[5];
float reales[15];

Iniciacion de un Vector en su definicion:

int numero[] = {1,4,6,7,3}; //un array o vector de numeros...de esta manera se podria rellenar un array o vector
numero[0]=1;
numero[1]=4;
numero[2]=6;
numero[3]=7;
numero[4]=3;//aca tambien se puede rellenar o determinar la posicion del numero,(en el caso de que sea un numero)

char letra[]={'a','e','i','o','u'};//array o vector de letras
letra[]='a';
letra[]='e';
letra[]='i';
letra[]='o';
letra[]='u';
//estos serian los vectores o array de una sola dimension
